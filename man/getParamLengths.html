getParamLengths {ParamHelpers}R Documentation

Return lengths of parameters in parameter set.

Description

Useful for vector parameters.

Usage

  getParamLengths(par.set)

Arguments

par.set

[ParamSet]
Parameter set.

Value

[integer]. Named and in same order as par.set.

Examples


ps <- makeParamSet(
  makeNumericParam("u"),
  makeIntegerParam("v", lower=1, upper=2),
  makeDiscreteParam("w", values=1:2),
  makeDiscreteVectorParam("x", len=2, values=c("a", "b"))
)
getParamLengths(ps)

## u v w x 
## 1 1 1 2

# the length of the vector x is 2, for all other single value parameters the length is 1.
